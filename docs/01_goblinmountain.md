ゴブリンの山砦 シナリオ
===============================================================================

冒頭
-------------------------------------------------------------------------------

街道でPCが歩いているとゴブリたちが襲いかかってくる

* コボルド × 2
* ゴブリン × 1

3Rすると、後ろから援護が来る(NPC リーバング)
リーバング : 丁寧な口調でしゃべる糸目の剣士

* 種族 : 人間
* 性別 : ♂
* 年齢 : 25
* 技能 : 
  * フェンサー : Lv2
  * コンジャラー : Lv2
  * スカウト : Lv2
* 言語 : 古代語、妖精語
* 自己紹介 : 大したものではございません。一介の役人ですから
* 経験点 : なし
* 報酬 : なし
* MP : 20

### 背景情報 ###

PCに尋ねられれば以下のことを答えてよい

* リーバングは王都から派遣された地方官吏
* シェリ村を含む西方地域の担当
* 地方官吏は1年ごとに王都での事務処理と地方巡視をくりかえす
* 今回平和なシェリ村に行くこととなったが蛮族がうろついていてびっくりした

リーバングは冒険者に対して特に悪い感情はなく、
行くあてがないのなら一緒にシェリ村に行こうと誘う

### シェリ村での買い物 ###

辺境の村なので食料以外の買い物はほぼできないと思って良い


シェリ村での歓待
-------------------------------------------------------------------------------

シェリ村ではリーバングとともに来たPCたちが歓待を受ける
といっても質素な村であるので、夕食は固いパンに野菜のスープ
客人ということで干し肉がスープに浮かんでいて、これを村人は
またとないごちそうとばかりに喜ぶのだった

村での食事が終わる頃、司祭のワンリーが村に伝わる昔話を語る
娯楽の少ない村ではこれが楽しみらしく、みんな目を輝かせながら聞いている

### シェリに伝えられる昔話 ###

今から300年前、偉大な古代文明アル・メナスは、
蛮族たちの侵攻大破局(ディアポリック・トライアンフ)によって滅ぼされた。
まず最初に嵐と地震、津波の天変地異が次々と文明を襲い、弱り果てた人々に蛮族の軍団が襲い掛かってきたのだった。
人族は滅亡の危機に瀕したかと思われた。しかしそこで我が国ミルドの偉大なる王ミルドヴェルムが
調和の剣ルミエルの霊力で王土ミルドランドを保護し、王土の中の人族だけは生き残ることができた。
今でも王土の外は蛮族が跋扈し天地は荒れる地獄である。
王国の民は王ミルドヴェルムに感謝するとともに、大地を耕し神に努め、前線で王土を守る兵士を支えるべきである。

リーバングは満足そうに聞いているが、PCたちは自分たちの知っている世界観とちがうことに気づく
夕食も終えて夜が更けるころ、何やら不測の事態が起きたらしく村人たちが浮足立つ
